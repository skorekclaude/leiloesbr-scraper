# Kwas fosforowy(V) - H3PO4 - Produkcja metodą ekstrakcyjną (mokrą)

Zrodlo: "Wytwarzanie kwasu fosforowego(V) oraz nawozow fosforowych i wieloskladnikowych" - Aleksandra Tomczak, MEN/ITeE Radom 2006

---

## 1. Nazwa metody i produktu

- **Produkt**: Kwas ortofosforowy(V) - H3PO4 (ekstrakcyjny kwas fosforowy - EKF)
- **Metoda**: Ekstrakcyjna dwuwodzianowa (metoda mokra)
- **Zasada**: Rozklad surowca fosforowego (fosforytu) mocnym kwasem nieorganicznym (H2SO4) i oddzielenie soli wapniowych od roztworu H3PO4
- Alternatywa: metoda termiczna (TKF) - drozsza, ale czystszy produkt

### Wlasciwosci H3PO4
- Bezbarwne cialo stale, t.top. = 43 C
- Silnie higroskopijne, dobrze rozpuszczalne w wodzie
- Stezony (~80%) - oleista ciecz
- Odwadniany -> kwas pirofosforowy H4P2O7 (bezbarwny, szklisty, t.top. = 610 C)
- Dlugie silne ogrzewanie -> kwas metafosforowy HPO3
- Tworzy 3 szeregi soli: ortofosforany (Na3PO4), diwodoroortofosforany (NaH2PO4), wodoroortofosforany (Na2HPO4)

### Zastosowanie H3PO4
- Nawozy mineralne (wiekszosc produkcji)
- Dodatki paszowe
- Chemia gospodarcza, sole techniczne
- Preparaty farmaceutyczne
- Srodki piorace
- Preparaty do odrdzewiania i usuwania kamienia kotlowego
- Dentystyka

## 2. Surowce

### 2.1. Mineraly fosforowe

#### Apatyt - Ca5(PO4)3(F,Cl,OH)
- Fosforan bezwodny, najliczniejsza grupa mineralow
- Krystalizuje w ukladzie **heksagonalnym**
- Postacie krysztalow: wydluzone slupkowe (do kilkuset kg), igielkowe (najczestsza), tabliczkowe
- Wystepuje w skupieniach ziarnistych i zbitych
- Odmiany wg anionu: fluoroapatyt Ca5(PO4)3F, chloroapatyt, apatyt hydroksylowy
- Moze zawierac jon weglanowy (apatyt weglanowy) + Na2O, Ce2O3 (do 5%), MgO, Fe2O3, Al2O3
- Lupliwosc wyrazna, przelam nierowny/muszlowy, kruchy
- Barwy: bialy, zielony, niebieski, zolty, szary, fioletowy, czerwony, bezbarwny
- Polysk szklisty

**Fluoroapatyt** Ca5(PO4)3F - najszerzej rozpowszechniony:
- 42,26% P2O5
- 3,78% F
- 50,3-55,9% CaO
- Rozpuszcza sie w mocnych kwasach nieorganicznych (HCl, HNO3, H2SO4)
- Odmiany weglanowe rowniez w slabych kwasach organicznych (np. cytrynowy)

**Zloza**: Polwysep Kola (Chibiny, b. ZSRR), Hiszpania, Indie, Wietnam

#### Fosforyty (fosfaty)
- Osadowe zloza apatytu
- Teoretycznie >50% fosforanow, ~30% P2O5
- Powstaly z wietrzejacych skal magmowych
- Formy: konkrecje (plytkie morze) lub warstwy (glebsze morze)
- **Zloza**: Maroko (60% zasobow swiatowych), USA, b. ZSRR, Algieria, Tunezja, Egipt, Izrael, Arabia Saudyjska, Chiny
- ~80% z kopaln odkrywkowych
- Niektore partie wzbogacone w uran
- Polska: niewielkie zasoby ubogich fosforytow (rejon Annopola) - produkcja oparta na imporcie
- Rozpuszczalnosc Ca3(PO4)2 w wodzie: 0,00486% (20 C) - bardzo mala

### 2.2. Wstepna obrobka fosforytow (mielenie)
- **Cel**: Zwiekszyc powierzchnie sumaryczna dla lepszego kontaktu z kwasem -> przyspieszenie reakcji
- **Wymagane rozdrobnienie**: 90% materialu < 0,16 mm (maczka fosforytowa)
- **Aparatura do mielenia**:
  - Mlyny pierscienowe trojwalcowe
  - Mlyny z powietrznym oddzielaniem maczki (zamiast przesiewania) + jednoczesne suszenie (fosforyty zawieraja do 10% wody)
- **Odsiewanie**: sita wibracyjne - material grubszy wraca do mlyna
- **Transport**: przenosnik slimakowy do wlasciwego procesu technologicznego

### 2.3. Kwas siarkowy(VI) - H2SO4
- Stosowany w stezeniu 96-98%, wstepnie rozcienczany
- ~50% produkcji siarki (w postaci H2SO4) idzie na H3PO4 i nawozy
- Surowce do produkcji H2SO4: siarka rodzima, siarczkowe rudy metali niezelaznych (Cu, Zn, Pb), siarkowodor, piryty, kwasy odpadowe, FeSO4

**Rozcienczanie H2SO4**:
- Zbiornik wylozony blacha olowiana lub winidurem
- Zbiornik posiada belkotke i wezownice olowiane
- Kolejnosc: najpierw odmierzona ilosc wody, potem H2SO4
- Mieszanie sprezonym powietrzem (belkotka)
- Proces **egzotermiczny** - czesto potrzebne chlodzenie
- Stezenie i temperatura rozcienconego H2SO4 zalezy od % P w surowcu fosforanowym

### 2.4. Gaz ziemny
- Paliwo kopalne, glowny skladnik: metan (>90%)
- Gaz suchy (samodzielne poklady) vs mokry (przy ropie/weglu)
- Sklad gazu suchego: CH4 94%, C2H6 1,2%, C3H8 0,7%, C4H10 0,4%, C5H12 0,2%, CO2 i inne 0,2%
- >90% wykorzystywany jako surowiec do otrzymywania wodoru

**Reakcje**:
```
CH4 + H2O -> CO + 3H2        (konwersja metanu)
3H2 + N2 -> 2NH3              (synteza amoniaku)
```

- NH3 -> nawozy azotowe i wieloskladnikowe (~80%), tworzywa sztuczne, mat. wybuchowe, hydrazyna, aminy, farmaceutyki (~20%)
- Zalety gazu ziemnego: nizsze zuzycie energii, nizsze naklady inwestycyjne, prostszy proces, nizsze koszty, ekologicznosc

### 2.5. Sol potasowa (KCl)
- Skladnik nawozow wieloskladnikowych NPK (zrodlo potasu)

## 3. Reakcje chemiczne

### 3.1. Glowna reakcja ekstrakcji (metoda dwuwodzianowa)

```
Ca10F2(PO4)6 + 10H2SO4 + 20H2O -> 6H3PO4 + 10CaSO4*2H2O + 2HF
```

Udzial P2O5 w apatycie:
```
Ca10F2(PO4)6 -> 3P2O5 + CaF2 + 9CaO
```

### 3.2. Metoda termiczna (TKF) - dla porownania

```
P4 + 5O2 -> 2P2O5                    (spalanie fosforu)
P2O5 + H2O -> 2HPO3                  (adsorpcja tlenku)
HPO3 + H2O -> H3PO4                  (uwodnienie)
```

### 3.3. Metoda dwustopniowa (spalanie ciekłego fosforu)

```
P4 + 5O2 -> 2P2O5
P2O5 + H2O -> 2HPO3
HPO3 + H2O -> H3PO4
```

### 3.4. Odsiarczanie EKF

```
Ca2+ + SO4(2-) -> CaSO4 (osad)
```

### 3.5. Produkcja superfosfatu prostego

Etap I - nadmiar H2SO4, tworzenie H3PO4:
```
2Ca5(PO4)3F + 7H2SO4 + 3H2O -> 7CaSO4 + 3Ca(H2PO4)2*H2O + 2HF
```
```
2Ca5(PO4)3F + 10H2SO4 + 5H2O -> 10(CaSO4)*0,5H2O + 6H3PO4 + 2HF
```

Etap II - H3PO4 reaguje z nierozlozonym fluoroapatytem:
```
Ca5(PO4)3F + 7H3PO4 -> 5Ca(H2PO4)2*H2O + HF
```

Reakcje uboczne z zanieczyszczeniami:
```
CaCO3 + H2SO4 -> CaSO4 + CO2 + H2O
MgCO3 + H2SO4 -> MgSO4 + CO2 + H2O
CaCO3 + 2H3PO4 -> Ca(H2PO4)2*H2O + CO2
```

Reakcje fluorowodoru z krzemionka:
```
4HF + SiO2 -> SiF4 + 2H2O
2HF + SiF4 -> H2SiF6
```

### 3.6. Produkcja nawozow NPK (pulpa MAP)

```
H3PO4 + NH3 -> NH4H2PO4          (diwodoroortofosforan amonu - MAP)
NH4H2PO4 + NH3 -> (NH4)2HPO4     (wodoroortofosforan amonu - DAP)
H2SO4 + 2NH3 -> (NH4)2SO4        (silnie egzotermiczna - utrzymuje temperature)
```

## 4. Etapy procesu technologicznego - produkcja EKF metoda ekstrakcyjna dwuwodzianowa

### Etap 1: Mielenie surowcow fosforowych

- **Co zachodzi**: Rozdrabnianie fosforytu do maczki fosforytowej
- **Warunki**: Wymagane 90% ziaren < 0,16 mm
- **Aparatura**:
  - Mlyny pierscienowe trojwalcowe LUB
  - Mlyny z powietrznym oddzielaniem maczki (jednoczesne suszenie - fosforyty do 10% wody)
  - Sita wibracyjne (odsiewanie)
- **Wejscie**: surowy fosforyt (bryly)
- **Wyjscie**: maczka fosforytowa (< 0,16 mm)
- **Transport**: przenosnikiem slimakowym do reaktorow

### Etap 2: Dozowanie kwasu siarkowego(VI) i poplyckek do reaktorow

- **Co zachodzi**: Rozcienczanie H2SO4 (96-98%) do odpowiedniego stezenia; dodanie popluczkek (scieki z przemywania fosfogipsu)
- **Warunki**: Stezenie i temperatura rozcienconego H2SO4 zaleza od % P w surowcu
- **Aparatura**:
  - Zbiornik wylozony blacha olowiana lub winidurem (twarde PCV)
  - Belkotka do mieszania sprezonym powietrzem
  - Wezownice olowiane do chlodzenia
- **Wejscie**: H2SO4 96-98%, woda, popluczkowe kwasy fosforowe
- **Wyjscie**: rozcienczony H2SO4 o odpowiednim stezeniu i temperaturze
- **Uwaga**: Proces rozcienczania jest **egzotermiczny** - wymaga chlodzenia!
- Do rozcienczania najlepiej uzyc rozcienconego H3PO4 z przemywania fosfogipsu (uzyskuje sie bardziej stezony EKF)

Mozna tez dozowac kwas pohydrolityczny (odpad z prod. bieli tytanowej) - zaleznosc od jakosci i zanieczyszczen (Fe+Al+Mg). Max zanieczyszczen: 0,75 kmol/t 100% H3PO4.

### Etap 3: Dozowanie surowca fosforowego do reaktorow

- Dozowanie sprezone z dozowaniem czystego H2SO4
- Rownomierne dozowanie - kluczowe dla wydajnosci

### Etap 4: Ekstrakcja (wezel ekstrakcji)

- **Co zachodzi**: Rozklad fosforytu kwasem siarkowym -> H3PO4 + fosfogips (pulpa)
- **Reakcja glowna**: Ca10F2(PO4)6 + 10H2SO4 + 20H2O -> 6H3PO4 + 10CaSO4*2H2O + 2HF
- **Warunki**:
  - Stosunek fazy cieklej do stalej w pulpie: **3:1** (dostateczna plynnosc do mieszania i filtrowania)
  - Stabilna temperatura w reaktorach
  - Nadmiar H2SO4
  - Stale stezenie P2O5 w pulpie
  - Ciagle dozowanie srodkow przeciwpianowych
- **Aparatura**:
  - **Reaktory**: Stal pokryta warstwa gumy, wymurowane ksztaltkami kwasoodpornymi, mieszadla ze stali kwasoodpornej
  - **Wyparka adiabatyczna**: Do chlodzenia i zageszczania pulpy, wykonana z tych samych materialow co reaktory
  - Uklad pracuje z **duzym recyklem pulpy**
- **Materialy konstrukcyjne**: Guma + ksztaltki kwasoodporne + stal kwasoodporna (ze wzgledu na korodujace H2SO4 i HF)
- **Wejscie**: maczka fosforytowa, rozcienczony H2SO4, popluczkowe roztwory H3PO4
- **Wyjscie**: pulpa (zawiesina CaSO4 w H3PO4), gazy fluorowe (HF, SiF4)

**Czynniki wplywajace na wydajnosc ekstrakcji**:
1. Stopien rozdrobnienia surowca
2. Rownomierne dozowanie surowcow, prawidlowa proporcja fosforyt:H2SO4
3. Stabilne parametry (temperatura, nadmiar H2SO4, stezenie P2O5)
4. Prawidlowa recyrkulacja pulpy
5. Dobre wymieszanie pulpy
6. Staly poziom pulpy i stezenia CaSO4 w reaktorach
7. Ciagle dozowanie srodkow przeciwpianowych
8. Stabilnosc ruchu calej instalacji

### Etap 5: Chlodzenie pulpy reakcyjnej

- **Aparatura**: Wyparka adiabatyczna
- Chlodzenie i jednoczesne zageszczanie pulpy
- Materialy: guma + ksztaltki kwasoodporne (jak reaktory)

### Etap 6: Absorpcja gazow fluorowych z wezla ekstrakcyjnego

- **Co zachodzi**: Pochlaniane HF i SiF4 w roztworze kwasu fluorokrzemowego
- **Aparatura**: Absorber fluoru
- **Ciecz absorpcyjna**: roztwor kwasu heksafluorokrzemowego H2SiF6
- **Wejscie**: opary z reaktorow (HF, SiF4, CO2, pyly)
- **Wyjscie**: oczyszczone gazy -> komin; roztwor H2SiF6 (do 15% -> na Na2SiF6 lub 20% -> kwas handlowy)

### Etap 7: Filtracja pulpy (wezel filtracji)

- **Co zachodzi**: Rozdzielenie pulpy na faze stala (fosfogips) i ciekla (H3PO4)
- **Aparatura**: Filtr (pulpa wprowadzana z ukladu reaktorow)
- **Przemywanie**: Fosfogips przemywa sie **trzykrotnie woda**
- **Wejscie**: pulpa reakcyjna
- **Wyjscie**:
  - Kwas fosforowy(V) surowy (~25% P2O5) -> do magazynu
  - Fosfogips -> na haldy (skladowisko)
  - Popluczkowe H3PO4 -> z powrotem do ekstrakcji (recyrkulacja)

### Etap 8: Magazynowanie kwasu fosforowego(V) surowego

- Produkt: kwas o stezeniu ok. 25% P2O5

### Etap 9: Ewakuacja fosfogipsu

- Odprowadzanie tasmoiciagiem na skladowisko
- Srednia ilosc: 4-5 t fosfogipsu (sucha masa) / t 100% H3PO4

### Etap 10: Zatezanie kwasu surowego (wezel zatezania)

- **Co zachodzi**: Odparowanie czesci wody + wydzielanie oparow ze zwiazkami fluoru
- **Cel**: Zwiekszyc stezenie z ~25% P2O5 do **32-54% P2O5** (w zaleznosci od zastosowania)
- **Warunki**: Praca **pod proznia**, ogrzewanie para
- **Mechanizm**: Rozprezanie podgrzanego w wymienniku ciepla roztworu H3PO4 -> odparowanie wody
- **Aparatura**:
  - Szereg **wyparek zatezajacych** pracujacych pod proznia
  - Stalowy korpus **wygumowany**
  - Urzadzenia wewnetrzne (rurki grzejne, rura przelewowa, przegrody) ze **stali kwasoodpornej**
  - Wymiennik ciepla (podgrzewanie kwasu przed rozprezaniem)
- **Wejscie**: surowy H3PO4 (~25% P2O5), para grzewcza
- **Wyjscie**: zatezony H3PO4 (32-54% P2O5), opary fluorowe

### Etap 11: Absorpcja gazow fluorowych z wezla zatezania

- Opary oczyszczane w oddzielnym wezle absorpcji
- Podobny mechanizm jak w etapie 6

### Etap 12: Oczyszczanie EKF ze zwiazkow fluoru (odfluorowanie)

EKF zawiera znaczne ilosci domieszek, szczegolnie zwiazkow fluoru.

**Dwuetapowe odfluorowanie**:
1. **Etap ekstrakcji**: Dozowanie lugu sodowego (NaOH) do reaktora -> wytraczenie fluoru jako **Na2SiF6** (heksafluorokrzemian sodu)
2. **Etap zatezania**: Dozowanie krzemionki (SiO2) do surowego H3PO4 -> fluor przechodzi do fazy gazowej -> absorpcja w wezle absorpcji

### Etap 13: Odsiarczanie EKF (opcjonalne)

- Problem: ~4% siarczanow uniemozliwia produkcje technicznych soli fosforowych
- Wykorzystanie ograniczonej rozpuszczalnosci CaSO4 w H3PO4
- Reakcja: Ca2+ + SO4(2-) -> CaSO4 (osad)

### Etap 14: Klarowanie kwasu fosforowego(V) odfluorowanego

- **Problem**: Zatezony H3PO4 (52% P2O5) nie jest klarowny - zawiera sole nieorg., zawiesiny Ca/Fe/Al, nieprzereagowana krzemionka, krysztaly Na2SiF6
- **Metoda tradycyjna**: Naturalna **sedymentacja** w odstojnikach (kilkanascie dni!)
- **Przyspieszone klarowanie**: Dodatek **flokulanta** -> czas opadania: kilka godzin
- Zawiesina splywa grawitacyjnie do zbiornikow magazynowych
- W zbiornikach czesc soli wytraca sie i sedymentuje
- **Szlam** z dna zbiornika odpompowuje sie do reaktorow (recyrkulacja)
- Sklarowany kwas -> na wydzial nawozowy

### Etap 15: Magazynowanie kwasu zatezanogo

- Zbiorniki magazynowe
- Zbiornik rozdzielczy kwasu zatezanego -> rozsylanie do magazynow
- Po odstawianiu i odprowadzeniu szlamu -> kwas gotowy do wysylki

## 5. Schematy i diagramy (opisane w tekscie)

### Rys. 3: Schemat ideowy produkcji H3PO4 (metoda ekstrakcyjna)
Ciag technologiczny:
```
Fosforyt -> [Mielenie] -> Maczka fosforytowa
H2SO4 -> [Rozcienczanie + H2O] -> Rozcienczony H2SO4
                |                        |
                v                        v
            [EKSTRAKCJA (reaktory)] <-- popluczkowy H3PO4 (recyrkulacja)
                |           |
                v           v
        [gazy fluorowe]   [PULPA]
            |               |
            v               v
    [ABSORPCJA]      [FILTRACJA]
        |               |         |
        v               v         v
    H2SiF6        H3PO4 surowy   Fosfogips -> halda
                    |
                    v
              [ZATEZANIE (wyparki prozniowe)]
                    |          |
                    v          v
            H3PO4 zatezony  [gazy fluorowe -> ABSORPCJA]
            (32-54% P2O5)
```

### Rys. 4: Mozliwosci gospodarczego wykorzystania fosfogipsu
- Zrzuty do wod morskich i srodladowych
- Wykorzystanie efektywne (rolnictwo, budownictwo, inne)
- Skladowanie na ladzie

### Rys. 5: Mozliwosci rolniczego wykorzystania fosfogipsu
- Srodek kondycjonujacy glebe
- Wytwarzanie mieszanek nawozowych (blending)
- Wytwarzanie nawozow zawiesinowych
- Wytwarzanie nawozow stalych (na bazie azotanu amonu lub mocznika)

### Rys. 7: Schemat ideowy produkcji superfosfatu pojedynczego (forma pylista)
```
Magazyn fosforytow -> [Rozdrabnianie] -> Zmielony fosforyt
H2SO4 -> [Rozcienczanie + H2O] -> [Chlodzenie + H2O] -> Rozcienczony H2SO4
                                                            |
                                                            v
                                         [Rozklad fosforytow] <- Zawiesina SiO2
                                          |              |
                                          v              v
                                    [Gazy odpadowe]   [Produkt]
                                          |              |
                                          v              v
                                  [Kolumna absorpcyjna] [Dojrzewanie produktu]
                                    + H2O    |              |
                                          v              v
                                    [Magazyn H2SiF6]  [Pakowanie]
                                    -> Prod. kriolitu     |
                                    [Komin -> atmosfera]  v
                                                   [Prod. nawozow granulowanych]
```

### Rys. 9: Schemat wytwarzania nawozow NPK na bazie czesciowo rozlozonego fosforytu
```
Zmielony fosforyt + H2SO4 + siarczan amonu + KCl
                    |
                    v
               [Reakcja / Granulacja]
                    |
                    v
               [Suszenie]
                    |
                    v
               [Klasyfikacja]
              /      |        \
    podziarno    produkt    nadziarno
    (wraca)                [Kruszenie] -> wraca
```

### Rys. 10: Powiazania surowce - polprodukty - produkty przemyslu nawozowego
Glowne powiazania:
- Siarka + powietrze -> Wytworna H2SO4
- H2SO4 + fosforyty -> Wytworna H3PO4 -> H3PO4 + H2SiF6 (odzysk fluoru)
- H2SO4 + fosforyty -> Wytworna SSP (superfosfat pojedynczy)
- H3PO4 + fosforyty -> Wytworna TSP (superfosfat potrojny)
- Gaz ziemny + powietrze + woda -> Wytworna NH3
- NH3 + H3PO4 -> Wytworna MAP/DAP
- NH3 + H2SO4 -> Wytworna siarczanu amonu
- MAP/DAP + KCl + inne -> Wytworna nawozow kompleksowych NPK
- Nawozy pojedyncze + KCl -> Mieszalnia nawozow -> Nawozy mieszane NPK

## 7. Produkty uboczne i odpady

### 7.1. Fosfogips (glowny odpad)

- **Sklad**: CaSO4*2H2O (94-96%), Ca3(PO4)2 (1-3%), CaF2 (1-3%), SiO2 (0,5-5%)
- **Postac**: Sol krystaliczna, barwa bialo-szara
- **Wilgoc**: 20-30%, pH min. 2,6
- **Ilosc**: 4-5 t (sucha masa) / t 100% H3PO4
- **Zanieczyszczenia**: P2O5 calkowite 0,6-2%, Fe2O3, Al2O3, metale ciezkie (Cd 3-13 ppm, Cr 1-17 ppm, Pb 1-20 ppm, Zn 6-17 ppm, Cu 2-10 ppm, Ni 1-9 ppm, Mn 1-9 ppm, As <1 ppm, Hg <1 ppm)
- **Odprowadzanie**: Tasmociag na skladowisko
- **>95% trafia na skladowiska** (bardzo male wykorzystanie gospodarcze)

Zastosowanie fosfogipsu:
- Nawoz siarczanowy (rejony nieprzemyslowe, niskie siarczany w glebie)
- Podloze pod uprawy pieczarek
- Srodek kondycjonujacy glebe
- Surowiec do mieszanek nawozowych
- Nawozow zawiesinowych i stalych (na bazie NH4NO3 lub mocznika)

### 7.2. Zwiazki fluoru (produkt uboczny)

Uwalniajace sie podczas ekstrakcji i zatezania w postaci HF i SiF4:
- Absorpcja w roztworze H2SiF6 (kwas heksafluorokrzemowy)
- Stezenie H2SiF6: do 15% -> na wytworne Na2SiF6, lub 20% -> kwas handlowy

**Zagospodarowanie zwiazkow fluoru**:
- Kwas fluorowodorowy (HF) - trawienie szkla, synteza fluorkow
- Kriolit (Na3AlF6) - topnik w przemysle szklarskim, aluminiowym, ceramicznym
- Heksafluorokrzemian sodu (Na2SiF6) - szklo mleczne, emalie, kity, betony kwasoodporne, fluorowanie wody

### 7.3. Scieki

- **Pochladnicze niezakwaszone**: nie wymagaja neutralizacji -> kanalizacja
- **Kwasne** (z filtracjiabsorpcji, zatezania): zawieraja H3PO4 i H2SiF6
  - Odprowadzane do oczyszczania i **neutralizacji mleczkiem wapiennym**
  - Osady -> na skladowisko fosfogipsu

### 7.4. Gazy

- Z ekstrakcji, filtracji i zatezania
- Sklad: HF, SiF4, H2SiF6, CO2, pyly surowcow
- Kierowane do stacji absorpcji fluoru -> po oczyszczeniu -> komin do atmosfery
- Bardzo niekorzystny wplyw na srodowisko!

### 7.5. Opary znad filtra
- Usuwane wentylatorem

## 8. Materialy konstrukcyjne aparatury (WAZNE!)

### Reaktory ekstrakcyjne:
- **Korpus**: Stal pokryta warstwa gumy
- **Wymurowanie**: Ksztaltki kwasoodporne
- **Mieszadla**: Stal kwasoodporna
- **Powod**: Silne dzialanie korodujace H2SO4 i HF

### Wyparki (zatezanie i adiabatyczne):
- **Korpus**: Stalowy, **wygumowany**
- **Wnetrze**: Rurki grzejne, rura przelewowa, przegrody - ze **stali kwasoodpornej**

### Zbiorniki na kwas (rozcienczanie H2SO4):
- **Wylozenie**: Blacha olowiana lub winidur (twarde PCV) lub teflon lub inne tworzywo kwasoodporne
- **Wezownice**: Stal kwasoodporna lub olow (olow juz nie stosowany w nowych instalacjach)

### Etap rozkladu fosforytow (dodatkowe zabezpieczenie):
- **Pokrycie**: Guma, faolit (zywica fenolowo-krezolowa z napelniaczem kwasoodpornym) lub teflon
- **Powod**: HF uwalniajacy sie obok kwasow

### Ogolna zasada:
- Wszystko co ma kontakt z H2SO4, H3PO4, HF musi byc kwasoodporne
- Guma, faolit, teflon, winidur, stal kwasoodporna - podstawowe materialy

## 9. BHP i bezpieczenstwo

### Zagroenia od zwiazkow fluoru (tabela 5):

**Fluorowodor (HF)**:
- Powazne oparzenia, zapalenie spojowek, moze doprowadzic do slepoty
- Uszkadza drogi oddechowe, watrobe, nerki, mozg
- Wypiera wapn z organizmu -> fluoroza
- Wywoluje hipokalcemie z tezyczka, dzialanie kardiotoksyczne
- Trujacy dla ryb i planktonu
- Szkodliwy dla roslin (znaczne szkody w drzewostanie i roslinach uprawnych)
- Korozja metali, niszczenie szkla

**Tetrafluorek krzemu (SiF4)**:
- Dziala drazniaco na drogi oddechowe

**Kwas heksafluorokrzemowy (H2SiF6)**:
- Silnie toksyczny
- Powoduje raka skory i blon sluzowych
- Pary i ciecz dzialaja zraco na wszystkie czesci ciala
- Podraznienia gardla i pluc, przewlekle niezyt krtani i drog oddechowych -> obrzek

### Srodki bezpieczenstwa:
- Praca w rekawicach gumowych, okularach typu "gogle", fartuchu ochronnym
- Stezone kwasy - praca pod dygestorium
- Przestrzeganie regulaminow pracowni, przepisow BHP i instrukcji ppoz.
- Monitoring zanieczyszczen (szczegolnie zwiazki fluoru w gazach odlotowych)
- Srodki przeciwpianowe - ciagle dozowanie
- Neutralizacja sciekow kwasnych mleczkiem wapiennym

### Emisja maksymalna z produkcji nawozow NPK:
- Zwiazki fluorowe: 0,15 kg/t
- Amoniak: 0,30 kg/t
- Tlenki azotu, CO, CO2: parametry nienormowane

## 10. Nawozy fosforowe - rodzaje i produkcja

### Rodzaje nawozow fosforowych:
| Nawoz | Skladnik glowny | Min. P2O5 |
|---|---|---|
| Superfosfat prosty | Ca(H2PO4)2 + CaSO4 | 16% |
| Superfosfat wzbogacony | Ca(H2PO4)2 + CaSO4 | 25% |
| Superfosfat potrojny | Ca(H2PO4)2 + CaSO4 | 38% |
| Precypitat (dwufosfat) | CaHPO4 | 38% |
| Fosforyty czesciowo rozlozone | Ca(H2PO4)2 + CaSO4 | 20% |
| Fosforyty miekkie | Ca3(PO4)2 + CaCO3 | 25% |
| Termofosfat | zasadowy fosforan Ca + krzemian Ca | 25% |
| Fosforan glinowo-wapniowy | ortofosforany Al i Ca | 30% |

### Produkcja superfosfatu - etapy:
1. **Przygotowanie surowcow**: Rozdrobnienie fosforytow do < 0,16 mm
2. **Rozcienczanie i chlodzenie H2SO4**: Do odpowiedniego stezenia i temperatury (30-40 C dla fosforytow, 50-70 C dla apatytow)
3. **Zarabianie superfosfatu**: Wymieszanie skladnikow, zapoczatkowanie rozkladu fosforytow
4. **Dojrzewanie**: W magazynie, 2-3 tygodnie - dokonczenie reakcji
5. **Absorpcja gazow fluorowych**: Rozpuszczenie w wodzie -> H2SiF6

### Superfosfat wzbogacony/potrojny:
- Fosforyty rozkladane H3PO4 (min. 50% H3PO4) zamiast H2SO4
- Mozna uzyskac nawoz o wysokiej zawartosci P nawet z ubogich fosforytow

## 11. Nawozy wieloskladnikowe NPK

### Podzial:
- **Mieszane**: mechaniczne zmieszanie skladnikow
- **Zlozone (kompleksowe)**: wytworzone droga reakcji chemicznych
- **Plynne**: klarowne (RSM) i zawiesinowe

### Produkcja granulowanych NPK z H3PO4 - etapy:
1. **Wytworzenie pulpy MAP** w saturatorze: H3PO4 + NH3 -> NH4H2PO4, + H2SO4 + 2NH3 -> (NH4)2SO4 (utrzymanie temperatury)
2. **Zmieszanie pulpy MAP** z zawrotem + KCl + MgCO3 + wypelniacze + mikroelementy (MnO, ZnO, CuO, Na2B4O7*5H2O) w blizniacych **mieszalnikach-granulatorach**
3. **Suszenie granulatu**: goruce powietrze + spaliny z gazu ziemnego (temp. zalezy od skladu)
4. **Sortowanie**: Wezel frakcjonowania; prawidlowa wielkosc: **2-5 mm**; za male/duze -> powrot na poczatek
5. **Chlodzenie**: Powietrzem
6. **Powlekanie**: Natryskiwanie srodkiem antyzbrylajacym (np. wodna emulsja parafinowa)
7. **Absorpcja oparow i pylow**: Cyklony + pluczkki -> ograniczenie emisji NH3 i pylu. Pyl wraca do procesu
8. **Magazynowanie** produktu koncowego

### Produkcja NPK z czesciowo rozlozonego fosforytu:
1. Odmierzanie i wstepne zmieszanie surowcow
2. Rozdrabnianie i homogenizacja
3. Granulowanie (zwilzanie NH3, H3PO4 lub para wodna)
4. Suszenie (powietrze + spaliny gazu ziemnego)
5. Segregacja (podziarno -> granulator, nadziarno -> kruszenie -> granulator)
6. Schladzanie + pokrycie srodkiem antyzbrylajacym
7. Magazynowanie
